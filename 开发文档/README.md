## 文档维护规范

- 任何**接口/字段/表**变更都需要更新对应文档,变更随 PR 一起 review。
- 各文档头部带「最后更新」字段,修改时务必同步。
- 重大版本(从 V1 → V2 → V3)新增单独的「附录:版本差异」小节,而不是改原文。
- 文档使用纯 Markdown,不依赖第三方扩展;图表用 ASCII / Mermaid(后续可加)。
